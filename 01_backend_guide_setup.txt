Step 7 вЂ” Create Student Model

Open:

backend/api/models.py

Add:

from django.db import models

class Student(models.Model):
    name = models.CharField(max_length=100)
    course = models.CharField(max_length=100)
    age = models.IntegerField()

    def __str__(self):
        return self.name
Step 8 вЂ” Make Migrations

Step 9 вЂ” Create Serializer

Create a new file:

backend/api/serializers.py

Add:

from rest_framework import serializers
from .models import Student

class StudentSerializer(serializers.ModelSerializer):
    class Meta:
        model = Student
        fields = '__all__'
Step 10 вЂ” Create API Views

Open:

backend/api/views.py

Add:

from rest_framework import viewsets
from .models import Student
from .serializers import StudentSerializer

class StudentViewSet(viewsets.ModelViewSet):
    queryset = Student.objects.all()
    serializer_class = StudentSerializer
Step 11 вЂ” Set Up API URLs

Open:

backend/server/urls.py

Replace the default code with:

from django.contrib import admin
from django.urls import path, include
from rest_framework.routers import DefaultRouter
from api.views import StudentViewSet

# Create DRF router and register viewsets
router = DefaultRouter()
router.register(r'students', StudentViewSet)

# Combine router URLs with other URL patterns
urlpatterns = [
    path('admin/', admin.site.urls),
    path('api/', include(router.urls)),  # include the router URLs
]
